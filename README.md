# Lab5SecData

Este laboratorio se enfoca en el analisis de trafico de red usando Scapy, estadistica descriptiva y tecnicas de deteccion de anomalias. A partir de una captura .pcap, se examinan paquetes, se identifican patrones sospechosos con z-score e Isolation Forest, y finalmente se revisa manualmente el payload para entender mejor el comportamiento detectado.
